# Modern Warfare 2019 Community Consensus Pass

Issue scope: #15, fan/community consensus rankings only.

This pass uses the end-of-title r/CoDCompetitive aggregate survey result as the v1 scored source. This is a strong v1 backbone source for #15 because it is an end-of-title annual community survey with a published top-30 result image.

## Verdict

The reliable v1 Modern Warfare 2019 community consensus source set is:

- [The top 30 players in Modern Warfare according to r/CodCompetitive](https://www.reddit.com/r/CoDCompetitive/comments/im3gkx/the_top_30_players_in_modern_warfare_according_to/) - scored aggregate survey; `coverage: contemporaneous`; `season_coverage: end_of_title`; `ranking_size: 30`.

Reviewed but not scored: Breaking Point/player/media lists, stat-ranking posts, CDL awards, CoD Esports Wiki/history pages, Esports Earnings/results pages, and general discussion threads that did not add clean independent community ballots beyond the annual survey.

## Caveats

- Blanks or zeros in event wins are local source-data results, not claims that the player had no meaningful season impact.
- Future work can add independent thread clusters if we find clean end-of-title community ballots that are not just reactions to the same annual survey.
