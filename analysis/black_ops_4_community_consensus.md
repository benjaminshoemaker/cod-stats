# Black Ops 4 Community Consensus Pass

Issue scope: #15, fan/community consensus rankings only.

This pass uses the end-of-title r/CoDCompetitive aggregate survey result as the v1 scored source. This is usable as a v1 source, but it is weaker than the later annual surveys because the chart is top-10 inclusion frequency rather than an ordinal top-30 score.

## Verdict

The reliable v1 Black Ops 4 community consensus source set is:

- [The top 10 players in Black Ops 4 according to r/CodCompetitive](https://www.reddit.com/r/CoDCompetitive/comments/cv8yev/the_top_10_players_in_black_ops_4_according_to/) - scored aggregate survey; `coverage: contemporaneous`; `season_coverage: end_of_title`; `ranking_size: 30`.

Reviewed but not scored: Breaking Point/player/media lists, stat-ranking posts, CDL awards, CoD Esports Wiki/history pages, Esports Earnings/results pages, and general discussion threads that did not add clean independent community ballots beyond the annual survey.

## Interpretation

## Caveats

- The source measures top-10 inclusion agreement, not exact rank. The top four are useful consensus signals, but the middle/lower top 10 should be read as inclusion density.
- Blanks or zeros in event wins are local source-data results, not claims that the player had no meaningful season impact.
- Future work can add independent thread clusters if we find clean end-of-title community ballots that are not just reactions to the same annual survey.
