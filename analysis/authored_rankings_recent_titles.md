# Recent Title Authored Rankings Pass

Issue scope: #16, authored media/player/creator/caster/coach ranking evidence.

This pass mirrors the #15 community-consensus deep dives, but it does not score
or aggregate sources as community ballots. The current reliable backbone is the
Breaking Point Top 20 player list published through the Breaking Point Awards
pages for each recent title.

This pass excludes:

- r/CoDCompetitive annual surveys and thread clusters, which belong to #15;
- official awards, BP Rating awards, MVPs, and All-Star style honors, which
  belong to #11 if formal enough;
- objective K/D, KOR, damage, or mode-stat models, which belong to #13/#17.

## Verdict

The recent-title authored lane now has good first-pass coverage for:

- Modern Warfare 2019;
- Black Ops Cold War;
- Vanguard;
- Modern Warfare II;
- Modern Warfare III;
- Black Ops 6.

Each title has one strong seeded authored/media source: a Breaking Point Top 20
title-season ranking. That is enough for a source-backed evidence lane or
player-page cards, but not enough for a multi-source authored consensus score.

The best next source family is timestamped video/podcast/player/caster content
for the same titles. Those should remain verification leads until each has a
stable URL, timestamp, speaker identity, and transcript or hand-verified quote.

## Interpretation

Use these rows as an authored title-season evidence lane:

- "Breaking Point ranked this player #N for this title."
- "This is media/stat-outlet evidence, not fan consensus."
- "This is title-specific performance evidence, not all-time greatness."

Do not use these rows as:

- a community consensus score;
- a formal award/honor;
- an objective-stat model;
- an all-time player ranking.
