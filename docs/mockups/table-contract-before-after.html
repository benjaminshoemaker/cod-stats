What this would mean in production

Shared CSSIntroduce a table shell class: caption area, scroll hint, focusable overflow wrapper, optional sticky first column.
MarkupUse captions/labels and `scope` for plain tables; add equivalent ARIA/keyboard checks for Tabulator where semantic table markup is not available.
ScopeApply sticky first column only where horizontal scroll would hide row identity: leaderboard, compare, heatmap table, and wide player/game tables.
TestsAdd Playwright checks for no body overflow, focusable scroll region, scroll hint visibility, and sticky first column on mobile.
