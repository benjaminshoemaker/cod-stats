Player stats mockups

A proposed player-page treatment for objective skill stats: career summary, season breakdown, and tournament breakdown. Values shown are real Scump aggregates from the current slim PlayerStats source.

Current player page
Recommendation: yes, we should support both season and tournament stats. The generated player payload should add `skillStats.byGame[]` and `skillStats.byEvent[]`, while keeping the top-level career aggregate for quick scanning.
