Design Notes

Keep lanes separate

The page should not average consensus, K/D, MVPs, and wins into one magic score. The value is seeing where each lens disagrees.

Show sample size

Skill rows expose maps and events. A high K/D on 15 maps should not read like the same evidence as 80 maps.

Treat awards as sparse

Only Karma and Crimsix have formal BO2 Event MVP rows in the current source set. Everyone else is "not found here," not zero.
