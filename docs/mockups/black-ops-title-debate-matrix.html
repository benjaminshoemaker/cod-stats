Black Ops Title Debate Matrix

A mockup of a title-level evidence page using live data from site/data.js. It keeps resume, placement, recorded skill, and formal honors in separate lanes so early-era source gaps stay visible.

Current season page
Important early-era caveat. The original Black Ops season has rich major/result data in this repo, but the current source set has no usable PlayerStats or formal honors rows for this title. The mockup renders those lanes as unavailable, not zero.

What This Mockup Proves

It works even when lanes are sparse

Black Ops is a hard case because stats and honors are unavailable. The page still says something real: TeeP leads the resume lane, while MerK and BigTymeR look strongest by average placement among the main winners.

It prevents fake certainty

The page can say “resume says X, placement says Y, skill/honors are unavailable” instead of pretending missing early-era evidence is proof against a player.
